Children-Learning App

Type:
C# WPF desktop application

Main file:
Children-Learning App.exe

Purpose:
This app helps children learn the alphabet from A to Z. Each letter includes an associated word, letter sound prompt, simple sentence, tracing practice, and exercises.

Features:
- A-Z alphabet navigation
- Letter lesson cards
- Associated word for every letter
- Example sentence for every letter
- Multiple-choice exercises
- Typing practice
- Letter tracing/drawing area
- Progress tracker
- Local progress saving

Saved progress:
Progress is saved locally on this computer under:
%LOCALAPPDATA%\GhoZTech\Children-Learning App\progress.json

No internet connection, database, website, or cloud account is required.
